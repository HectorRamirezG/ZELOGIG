# ZelogiG

ZelogiG es una experiencia web para aprender ingles resolviendo rompecabezas narrativos y logicos. Las palabras son piezas del mundo: al construir una frase valida, el estado del nivel cambia.

## Estado actual

Fase 0: fundacion estrategica y arquitectonica.

Todavia no se ha generado codigo funcional, SQL, componentes, endpoints ni instalaciones. Esta restriccion es intencional.

## Producto

- Plataforma: ZelogiG
- Primer modo de juego: Grammar Logic
- Propuesta: aprender haciendo, decidiendo y resolviendo problemas en contexto
- Experiencia: cercana a un videojuego narrativo, no a un curso tradicional
