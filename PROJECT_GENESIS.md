# PROJECT GENESIS

## Autoridad

Este archivo define la direccion estable de ZelogiG. Si una solicitud futura contradice este documento, debe detenerse y abrirse un RFC.

## Vision

ZelogiG ayuda a desarrollar comunicacion real en ingles mediante simulaciones y rompecabezas con consecuencias visibles. El aprendizaje sucede dentro de la experiencia; la explicacion aparece despues de la accion.

## Principios inmutables

1. Aprender haciendo.
2. Aprender tomando decisiones.
3. Aprender dentro de contexto.
4. La simulacion es el producto.
5. La IA amplifica, pero no reemplaza, el diseno del aprendizaje.
6. La comunicacion efectiva tiene prioridad sobre la memorizacion.
7. Toda funcion debe reforzar la inmersion.
8. Simplicidad extrema.
9. El sistema debe ser mantenible por una persona.
10. Las decisiones importantes deben quedar escritas.

## Arquitectura

- Next.js App Router, React, TypeScript y Tailwind CSS.
- Supabase para autenticacion, datos y funciones estrictamente necesarias.
- Vercel para publicacion.
- Monolito modular.
- Logica del dominio separada de la interfaz.

## Regla de proteccion

Rechazar o aplazar cualquier propuesta que aumente complejidad, rompa la arquitectura, cree deuda tecnica o desvie la experiencia principal.
