# Sistema de diseno

## Direccion visual

Soft-Tech: amable, aireado, tactil y sereno.

## Colores

- Lavanda para atmosfera y acciones suaves
- Rosa para elementos expresivos
- Menta para estados positivos
- Azul claro para informacion
- Neutros suaves para paredes y fondos

El color nunca sera el unico medio para comunicar un estado.

## Forma

- Bordes amplios
- Esquinas redondeadas
- Sombras suaves
- Espacio en blanco generoso
- Movimiento corto y funcional

## Accesibilidad

- Contraste verificable
- Foco visible
- Navegacion por teclado
- Texto alternativo o descripcion espacial
- Preferencia de movimiento reducido
- Bloques distinguibles por texto, patron o icono, no solo por color
