# Estrategia de pruebas

## Unidad

Validar movimiento, cadenas de empuje, limites, parser, reglas, victoria y deshacer.

## Integracion

Validar que el controlador actualiza mapa, historial, reglas y estado de victoria como una sola operacion.

## Recorrido completo

Validar inicio, apertura de nivel, movimientos, regla activa, victoria, reinicio y persistencia local.

## Regla de publicacion

Un nivel no se publica si su documento es invalido o si no cuenta con una solucion registrada por diseno.
