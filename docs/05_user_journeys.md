# Recorridos del usuario

## Primera sesion

1. Descubre una escena corta.
2. Aprende a moverse sin tutorial extenso.
3. Empuja una palabra.
4. Forma una regla sencilla.
5. Observa un cambio inmediato.
6. Resuelve el nivel.
7. Recibe una explicacion breve.

## Regreso

1. Continua desde el ultimo nivel.
2. Recuerda la mecanica mediante el propio escenario.
3. Encuentra una variacion con una nueva decision linguistica.
4. Revisa su progreso sin metricas abrumadoras.

## Error util

1. Forma una estructura invalida.
2. El mundo no cambia.
3. La interfaz senala discretamente el origen del fallo.
4. Tras el intento, se muestra una pista contextual.
5. El usuario vuelve a decidir.
