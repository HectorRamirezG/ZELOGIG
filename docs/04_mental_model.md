# Modelo mental

## Para el jugador

- El mundo es una cuadricula.
- Algunas piezas representan objetos.
- Otras piezas representan palabras.
- Las palabras contiguas pueden formar una regla.
- Una regla valida cambia el comportamiento del mundo.
- Una regla invalida no cambia el mundo y ofrece retroalimentacion posterior.

## Para el sistema

Entrada del usuario -> intento de movimiento -> validacion de colision -> nuevo mapa -> lectura de reglas -> aplicacion de efectos -> retroalimentacion -> evaluacion de victoria.

## Limite conceptual

La gramatica valida la accion, pero el objetivo es comunicar significado y resolver el problema.
