# Principios fundacionales

1. La accion precede a la explicacion.
2. Cada nivel plantea un problema comprensible.
3. El error debe informar sin castigar.
4. La correccion sucede despues del intento.
5. El jugador debe entender por que una frase funciono o fallo.
6. La interfaz desaparece para dejar hablar al mundo.
7. Cada dependencia debe justificar su costo.
8. Una sola persona debe poder mantener el producto.
9. Las reglas del dominio no viven dentro de componentes visuales.
10. La documentacion es memoria externa para personas e inteligencias artificiales.
