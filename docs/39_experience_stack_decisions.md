# Decisiones del sistema de experiencia

## Diseño

- Dirección propia de ZelogiG.
- CSS y SVG para el MVP.
- Sin biblioteca de componentes visuales genérica.

## Animación

- CSS nativo inicialmente.
- Biblioteca especializada solo ante complejidad demostrada.

## Audio

- Efectos breves mediante un adaptador propio.
- Música pospuesta.
- Recursos con procedencia y licencia registradas.

## Respuesta táctil

- API del navegador cuando esté disponible.
- Siempre opcional y con alternativa visual.

## Recursos generados con IA

- Útiles para conceptos.
- No entran directamente al producto sin normalización y revisión.
- No se imitan artistas vivos, marcas, personajes ni estilos protegidos.

## Móvil

- Interacción primaria mediante swipe.
- Controles equivalentes visibles y accesibles.
- Pruebas reales en dispositivos antes de publicar.

## Dependencias

Ninguna dependencia nueva queda autorizada por este documento. Toda incorporación se presenta con alternativas, peso, mantenimiento, licencia, costo, riesgo y plan de retiro.
