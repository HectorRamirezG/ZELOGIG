# Proceso RFC

Todo cambio importante usa `docs/rfc/RFC-XXX-titulo.md`.

## Contenido minimo

- Titulo
- Estado: pendiente, aprobado o rechazado
- Contexto y problema
- Motivacion
- Propuesta
- Alternativas consideradas
- Beneficios
- Riesgos
- Impacto en frontend
- Impacto en backend
- Impacto en base de datos
- Impacto en experiencia
- Impacto en costos
- Plan de reversa
- Decision y fecha

Sin RFC aprobado no se modifica la arquitectura.
