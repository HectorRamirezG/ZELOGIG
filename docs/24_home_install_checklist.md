# Lista para continuar en casa

1. Copiar la carpeta completa de ZelogiG.
2. Instalar Node.js LTS desde una fuente oficial.
3. Abrir una terminal nueva.
4. Confirmar `node --version`, `npm --version` y `git --version`.
5. No ejecutar instalaciones globales innecesarias.
6. Solicitar el instalador final de dependencias.
7. Ejecutar comprobacion de tipos, pruebas y servidor local.

No instalar Supabase CLI, Vercel CLI ni herramientas globales hasta que exista una necesidad concreta.
