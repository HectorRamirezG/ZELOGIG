# Identidad del producto

## Definición oficial

ZelogiG es un videojuego 2D de rompecabezas lingüísticos donde las palabras reescriben el mundo.

## Qué es

- Un juego de lógica.
- Una experiencia visual y táctil.
- Un sistema de reglas combinables.
- Una forma de descubrir estructuras del inglés mediante consecuencias.

## Qué no es

- Un simulador profesional.
- Un curso tradicional.
- Una lista de ejercicios.
- Una plataforma empresarial.
- Una copia visual o mecánica de una obra existente.

## Promesa

Mover palabras cambia significado. Cambiar significado transforma el mundo.
