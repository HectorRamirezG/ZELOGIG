# Anti features

No forman parte del MVP:

- Ranking global
- NFT o criptomonedas
- Marketplace
- Avatares 3D
- Chat entre usuarios
- Clanes o red social interna
- Multijugador
- Minijuegos desconectados del aprendizaje
- Gamificacion excesiva
- Arquitectura empresarial
- Analitica invasiva
- Vectores semanticos antes de demostrar su necesidad
- Integraciones externas sin un caso validado

Cualquiera de estos elementos requiere un RFC y evidencia de valor.
